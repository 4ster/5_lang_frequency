# Frequency Analysis of Words

This script loads text from file and prints top 10 frequent symbols with conunts from this text.

# Quickstart

Tu run on Linux, Python 3.5:

```bash

$ python lang_frequency.py <path to file>
' ': 584873
'о': 262804
'е': 188636
'а': 186189
'и': 154497
'н': 144144
'т': 133153
'с': 122433
'л': 114800
'в': 102592
```
